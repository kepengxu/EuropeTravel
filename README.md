# EuropeTravel

15-day low-intensity Italy, France, and Spain travel guide with route map, hotel zones, attraction ratings, route-aware restaurants, transport notes, reservations, and local images.

## GitHub Pages

This repository is configured with a GitHub Actions workflow at `.github/workflows/pages.yml`.

After pushing to `main`, enable Pages in GitHub:

1. Open the repository settings.
2. Go to `Pages`.
3. Under `Build and deployment`, choose `GitHub Actions`.
4. Wait for the `Deploy static travel guide to GitHub Pages` workflow to finish.

The site will publish at:

`https://kepengxu.github.io/EuropeTravel/`
